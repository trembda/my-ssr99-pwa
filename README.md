# SSR 99 PWA (prêt GitHub Pages)

## Contenu
- index.html (app)
- manifest.webmanifest (PWA)
- sw.js (cache/offline)
- icons/ (192px, 512px)

## Test local (obligatoire)
### Option A: VS Code + Live Server
Ouvre le dossier et "Go Live" sur index.html.

### Option B: Python
python -m http.server 5173
Puis ouvre: http://localhost:5173/

## Publication GitHub Pages
Repo -> Settings -> Pages -> Branch: main / folder: / (root)

## Installation
- Android (Chrome/Edge): "Installer" ou "Ajouter à l'écran d'accueil"
- iOS (Safari): Partager -> Ajouter à l'écran d'accueil
